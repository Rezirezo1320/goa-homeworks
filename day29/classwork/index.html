Front-end

რა არის Front-end-ი?

Front-end არის ვებსაიტის ან ვებაპლიკაციის ის ნაწილი, რომელსაც მომხმარებელი პირდაპირ ხედავს და იყენებს
ეს არის ვიზუალური და ინტერაქციული ნაწილი,

სად შევისწავლით Front-end-ს?

Front-end-ის შესწავლაა შეგვიძლია goal oriented academy-ში

რაში დაეხმარება Front-end-ი?

  1. Front-end-ის ძირითადი მიზანია ვებსაიტის ან აპლიკაციის იერსახის, ინტერფეისისა და ფუნქციონალის შექმნა.

  2. ვიზუალური ინტერფეისი: დიზაინისა და მომხმარებლის ინტერფეისის შექმნა,
    რაც ვებსაიტის ფუნქციონალურობას მარტივად აღქმადი და ლამაზი ხდის.

  3. ინტერაქტიულობა: JavaScript-ის დახმარებით, შეგიძლიათ შექმნათ დინამიური ელემენტები
    (მოძრავი მენიუები, ჩაშენებული ვიდეოები, ანიმაციები).

  4. მრავალპლატფორმიანობა: Front-end უზრუნველყოფს ვებსაიტის ადაპტაციას სხვადასხვა მოწყობილობებისთვის
    (Responsive Design) — დესკტოპის, ტაბლეტისა და მობილურისთვის.

რა შედის Front-end-ში?

  1. HTML (HyperText Markup Language): სტრუქტურის შექმნის ენა, რომელიც
    განსაზღვრავს ვებსაიტის კონტენტს (ტექსტი, სურათები, ლინკები და ა.შ.).

  2. CSS (Cascading Style Sheets): დიზაინის ენა, რომელიც
    განსაზღვრავს ვებსაიტის გარეგნობას (ფერები, შრიფტები, განლაგება).

  3. JavaScript: პროგრამირების ენა, რომელიც ვებსაიტს ან აპლიკაციას ინტერაქტიულობას აძლევს
    (მაგალითად, ღილაკებზე დაჭერა, სლაიდშოუები, ფორმების შემოწმება და ა.შ.).
